# 👤 关于熊哥

## 🎯 项目目标

熊哥每日资讯站旨在为技术爱好者提供一个快速了解行业动态的平台。每天早上花 15 分钟，了解昨天发生了什么。

## 🔧 技术架构

```
┌─────────────────────────────────────┐
│         GitHub Repository          │
├─────────────────────────────────────┤
│  📝 Markdown 内容                   │
│  ⚙️  GitHub Actions 自动构建        │
│  🌐 GitHub Pages 托管              │
│  📖 Docsify 渲染                   │
└─────────────────────────────────────┘
```

## 🛠️ 使用工具

| 工具 | 用途 |
|------|------|
| Docsify | 文档站点生成 |
| GitHub Actions | CI/CD 自动化 |
| GitHub Pages | 静态托管 |
