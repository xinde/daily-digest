# 🤝 贡献指南

感谢您对熊哥每日资讯站的关注！欢迎贡献优质内容。

## 📝 提交内容

### 资讯格式要求

每条资讯请包含以下字段：

```markdown
### 文章标题

📌 **标题**: 原标题  
💬 **简介**: 1-2句话说明  
📝 **总结**: 
- 关键亮点1
- 关键亮点2
- 关键亮点3
🔗 **链接**: https://example.com
```

### 内容来源

- ✅ Hacker News 热门
- ✅ GitHub Trending/Stars
- ✅ 技术博客精选
- ✅ 开源项目推荐
- ❌ 广告内容
- ❌ 纯营销文章

## 🚀 提交流程

1. Fork 本仓库
2. 创建新分支: `git checkout -b daily/YYYY-MM-DD`
3. 在 `daily/` 目录创建当日资讯文件
4. 更新 `_sidebar.md` 添加当日链接
5. 提交 Pull Request

## 📋 命名规范

- 文件名格式: `YYYY-MM-DD.md`
- 例如: `2026-04-03.md`

## ✅ 检查清单

- [ ] 内容真实可信
- [ ] 链接可正常访问
- [ ] 格式符合要求
- [ ] 无错别字
- [ ] 更新了侧边栏

如有疑问，请提交 Issue 或联系熊哥！
